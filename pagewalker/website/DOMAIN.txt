Custom domain for Pagewalker (hosted on Vercel)
===============================================

1) Choose and register a name
   Use any registrar (Cloudflare, Namecheap, Porkbun, Google Domains, etc.).
   Check availability for names like: pagewalker.app, getpagewalker.com, pagewalker.io

2) Attach it in Vercel
   Vercel Dashboard → your Project (pagewalker) → Settings → Domains → Add
   Enter your root domain (example.com) and/or www.example.com.
   Vercel shows the exact DNS records (usually A for apex, CNAME for www).
   Wait until the domain shows "Valid" and HTTPS works.

3) Update these files in this folder to use YOUR final URL (https only)
   Replace every https://pagewalker.vercel.app with https://your-domain.tld
   - index.html          (meta og:url, link rel=canonical)
   - sitemap.xml         (all loc entries)
   - robots.txt          (Sitemap line)

4) Supabase (password reset / auth redirects)
   Dashboard → Authentication → URL configuration
   - Site URL: https://your-domain.tld
   - Redirect URLs: include https://your-domain.tld/auth/update-password
   (Keep the Vercel preview URL in Redirect URLs too if you still test on .vercel.app)

5) Google Play Console
   Store listing → Contact details / Website: https://your-domain.tld
   Privacy policy URL: https://your-domain.tld/privacy

6) Optional: remove or keep the *.vercel.app subdomain
   Vercel can serve both; set canonical + Play Store to the custom domain only.
